share-tags-api

From any website, get the metadata used by search engines and social media for display link previews

How to use:

Add websites domain to the API URL, separated by commas

Example: http://share-tags-api.onrender.com/you.com,codi.link,notvalidurl

Maximum 10 websites at a time

Note: the API only accept the full domain without protocols or extra symbols
✖ .../https://color.adobe.com
✖ .../color.adobe.com/
✖ .../color.adobe/
✔ .../color.adobe.com

What returns?

A JSON array with the following information:

  • The URL itself
  • Open Graph Protocol tags (e.g. "og:description")
  • Twitter Cards tags (e.g. "twitter:description")
  • SEO tags
  • HTML meta tags (title, description)
  • Favicon
  • error: An error message if there is one
